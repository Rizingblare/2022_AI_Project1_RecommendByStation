# 부산 지하철 1호선 역별 맛집 추천 시스템
## 개요
공공데이터와 Python 웹 크롤링을 활용한 추천 시스템

## 배경 및 기획의도
- 코로나 관련 규제 완화
- 시민들의 활동량 증가


- 잇따라 침체되었던 지역 상권의 활성화

## 기술스택
- Python
- Django

## 제공기능
|기능|설명|
|---|---|
|역정보|부산 지하철 1호선의 지하철 역정보 제공|
|맛집추천|역별로 산출한 평점에 기반한 맛집 순위 제공|
